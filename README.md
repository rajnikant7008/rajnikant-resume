## Build from Source

Requires TeX Live with `sourcesanspro`, `mdframed`, and `lmodern` packages.

```bash
# Human-readable
pdflatex rajnikant-resume.tex

# ATS version
pdflatex rajnikant-resume-ats.tex
```

On macOS with MacTeX:

```bash
/Library/TeX/texbin/pdflatex rajnikant-resume.tex
/Library/TeX/texbin/pdflatex rajnikant-resume-ats.tex
```
